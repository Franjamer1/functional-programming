Actividad 1: Validación de caracteres Objetivo: Desarrollar un algoritmo que utilice una estructura condicional para verificar si un carácter ingresado pertenece a un conjunto determinado de valores válidos, y mostrar un mensaje en función del resultado. Crea un programa que solicite al usuario el ingreso de un carácter. El algoritmo debe verificar si el carácter ingresado es 'S' o 'N'. Si el carácter corresponde a alguno de esos dos valores, se debe mostrar en pantalla el mensaje “CORRECTO”. En caso contrario, el mensaje deberá ser “INCORRECTO”. Importante: Por el momento, no es necesario validar el formato de entrada. Se asume que el usuario ingresará el carácter en mayúsculas.



Actividad 2: Clasificación de días según condición Objetivo: Diseñar un algoritmo que analice un valor textual ingresado por el usuario y lo clasifique según una condición específica, utilizando una estructura condicional doble. Elabora un programa que solicite al usuario ingresar un día de la semana. Luego, el algoritmo deberá determinar si ese día corresponde a una jornada de entrenamiento o no. Considerar que los días de entrenamiento son: LUNES, MARTES, MIÉRCOLES y JUEVES. Importante: Por el momento, no es necesario validar el formato de entrada (como uso de mayúsculas o tildes). Se asume que el usuario ingresará el nombre del día completo y en mayúsculas.



Actividad 3: Determinar si un número es par o impar Objetivo: Desarrollar un algoritmo que utilice una estructura condicional junto con una función matemática para analizar un número y clasificarlo como par o impar. Crea un programa que solicite al usuario ingresar un número entero. El algoritmo debe determinar si dicho número es par o impar, y mostrar un mensaje en pantalla indicando el resultado del análisis. Pista: Un número es par si al dividirlo por 2, el resto de la división es igual a 0. Para realizar esta verificación de forma eficiente, se recomienda investigar el uso de la función mod en PSeInt, que permite obtener el resto de una división entera.
